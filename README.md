# Meal HA

Composant Home Assistant pour la gestion des repas :

- Bibliothèque d'aliments
- Création de plats
- Planification dans le calendrier (plat principal + accompagnement)
- Gestion de plusieurs listes de courses

Compatible HACS.

## Installation

1. Copier le dossier `custom_components/meal_ha` dans votre dossier `custom_components` de Home Assistant.
2. Redémarrer Home Assistant.
